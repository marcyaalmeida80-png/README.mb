✅ Requisitos Funcionais (RF)

RF01 Exibir informações sobre desigualdade no acesso à tecnologia em Fortaleza.
RF02 Apresentar dados estatísticos sobre alfabetização digital.
RF03 Permitir a navegação entre diferentes seções do site.
RF04 Exibir gráficos e indicadores relacionados ao tema.
RF05 Disponibilizar conteúdos educativos sobre inclusão digital.
RF06 Permitir a consulta de dados por categoria.
RF07 Exibir informações de forma organizada e intuitiva.

⚙️ Requisitos Não-Funcionais (RNF)

RNF01 O sistema deve possuir interface responsiva para computadores e dispositivos móveis.
RNF02 As páginas devem carregar em até 3 segundos em condições normais de internet.
RNF03 O sistema deve ser desenvolvido utilizando HTML, CSS e JavaScript.
RNF04 O sistema deve apresentar informações de forma clara e acessível.
RNF05 O sistema deve funcionar nos principais navegadores modernos (Chrome, Firefox, Edge e Safari).
RNF06 O código deve ser organizado e documentado para facilitar a manutenção.
RNF07 O sistema deve garantir a disponibilidade das informações apresentadas.

Arquitetura do Sistema - Módulos

🎯 Projeto de Inclusão Digital

Este projeto foi desenvolvido com o objetivo de reunir informações sobre a desigualdade no acesso à tecnologia em Fortaleza.

Através de dados estatísticos, gráficos interativos e conteúdos educativos, o sistema contribui para conscientizar sobre a importância da inclusão digital.

🏗️ Conceitos de OOP

Herança: Reutilização de código através de classes base especializadas.

Encapsulamento: Proteção de dados sensíveis com propriedades privadas.

Polimorfismo: Diferentes implementações do mesmo método em classes filhas.

💡 Recursos Principais

  • ✓ Interface responsiva e intuitiva
  • ✓ Dados estatísticos em tempo real
  • ✓ Gráficos interativos
  • ✓ Conteúdo educativo organizado
  • ✓ Compatibilidade com múltiplos navegadores
  • ✓ Código bem documentado
